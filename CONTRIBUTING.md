## Reporting Issues

### Bug Reports

When reporting bugs, please include:

- **Description** - Clear description of the issue
- **Steps to reproduce** - Detailed steps to reproduce the bug
- **Expected behavior** - What you expected to happen
- **Actual behavior** - What actually happened
- **Environment** - Node version, OS, Cloudflare Workers version, etc.
- **Logs** - Any relevant error messages or logs

### Feature Requests

While we primarily accept bug reports, you may also:

- Suggest features through GitHub Issues
- Note that feature implementation is at the discretion of the maintainer
- Feature requests may not be implemented immediately

## Questions?

If you have questions:

1. Check the [documentation](./docs/)
2. Search existing issues
3. Create a new issue with the question label
